# Hayama NFC-e Retirada

Site para gerar recibos NFC-e para retirada no Hayama Sushi.

## Funcionalidades

- Adicionar itens manualmente (nome, quantidade, valor)
- Subtotal automático por item
- Total geral atualizado automaticamente
- Layout pronto para impressora térmica (58mm/80mm)
- Botão de imprimir

## Uso

1. Abra `index.html` no navegador
2. Digite o item, quantidade e valor unitário
3. Clique em "Adicionar Item" para cada item
4. Clique em "IMPRIMIR NFC-e" para imprimir o recibo na impressora térmica

## Observações

- Este site **não emite NFC-e oficial**, apenas gera recibo para retirada.
