# Hayama NFC-e Retirada

Este projeto é um site simples para gerar recibos NFC-e para retirada no Hayama Sushi.

## Funcionalidades

- Adicionar itens, quantidade e valor
- Calcula total automaticamente
- Formata para impressão em impressora térmica (58mm/80mm)
- Botão para imprimir diretamente

## Como usar

1. Abra `index.html` no navegador
2. Preencha os itens, quantidade e valor
3. Clique em "Adicionar Item"
4. Clique em "IMPRIMIR NFC-e" para imprimir na máquina térmica

## Observações

- Este site **não substitui NFC-e oficial** emitida pela SEFAZ.
- É apenas para controle interno e impressão de recibo de retirada.
